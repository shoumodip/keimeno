* Text Editor
[[./demo-alpha3.png]]

A WIP text editor in C.

* Installation

#+begin_src console
$ git clone https://github.com/shoumodip/keimeno
$ cd keimeno
$ make
#+end_src

* Dependencies
- [[https://www.gnu.org/software/libc/][GNU C library]]
- [[https://invisible-island.net/ncurses/][Ncurses]]
