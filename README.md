# Text Editor
![demo](./demo-alpha3.png)

A WIP text editor in C.

# Installation

```bash
$ git clone https://github.com/shoumodip/keimeno
$ cd keimeno
$ make
```

# Dependencies
- [GNU C library](https://www.gnu.org/software/libc/)
- [Ncurses](https://invisible-island.net/ncurses/)
